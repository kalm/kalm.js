# Security Policy

## Supported Versions

| Version | Supported          | Current-version compatible |
| ------- | ------------------ | -------------------------- |
| 6.1 (current)   | :white_check_mark: | :white_check_mark: |
| 6.0   | :white_check_mark: | :white_check_mark: |
| 5.x   | :white_check_mark: | :x: |
| 4.x   | :x:  | :x: |
| 3.x   | :x:  | :x: |
| < 3.x   | :x:  | :x: |

## Reporting a Vulnerability

Security vulnerabilities may be reported through github issues for this project: https://github.com/kalm/kalm.js/issues
